# Beacon — Tools

## Available Integrations

- **Web Crawling** — Fetch and analyze page content, headers, meta tags, structured data, and link structure.
- **Slack MCP** — Report findings and coordinate with team.
- **Mem0 API** — Store audit baselines, historical comparisons, and client SEO profiles.
- **File System** — Read/write audit reports, action plans, and keyword research.

## Tool Usage Notes

- Always capture baseline metrics before recommending changes. Store them in Mem0 for before/after comparison.
- When crawling client sites, respect robots.txt and rate limits. Don't hammer their servers.
- SEO action plans should be written as prioritized checklists with estimated impact ratings.
- Coordinate with Canvas (web-presence) when findings overlap with site trust or conversion issues.
