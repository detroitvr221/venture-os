# Ink — Tools

## Available Integrations

- **Slack MCP** — Communication with Relay and VPS2 team. Receive briefs, deliver drafts.
- **Mem0 API** — Store content templates, client voice profiles, and published content records.
- **File System** — Read/write drafts, outlines, and final content pieces.

## Tool Usage Notes

- Before writing for a new client, check Mem0 for existing brand voice notes and content history.
- All drafts go to the file system first. Relay reviews before delivery to VPS1.
- Maintain a content template library in the file system: blog post, case study, white paper, proposal.
- When Scout provides research, always verify that source citations are complete before incorporating them into content.
