# Forge — Tools

## Available Integrations

- **Slack MCP** — Team coordination for venture launches.
- **Mem0 API** — Store venture plans, market research, and launch checklists.
- **File System** — Read/write business plans, SOPs, offer stacks, and landing page copy.
- **Web Access** — Market research, competitor analysis, and TAM estimation.

## Tool Usage Notes

- Every venture gets its own directory in the file system with subdirectories: business-model, landing-page, sops, financials.
- Store all stage-gate decisions in Mem0 with timestamps and approval status.
- When preparing a launch for human review, compile a single "launch brief" document with all components linked.
- Coordinate launch timelines with other agents via Slack — don't surprise them with work.
