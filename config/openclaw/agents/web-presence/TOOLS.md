# Canvas — Tools

## Available Integrations

- **Web Crawling** — Fetch and analyze site pages, layout, trust signals, and conversion elements.
- **Slack MCP** — Report audit findings and coordinate with team.
- **Mem0 API** — Store brand audit histories and client web presence profiles.
- **File System** — Read/write audit reports and recommendation documents.

## Tool Usage Notes

- When auditing, check website, Google Business Profile, and at least 2 directory listings per client.
- Store before/after snapshots in Mem0 so improvements can be tracked over time.
- Coordinate with Beacon (SEO) when technical performance issues affect web presence.
- Include specific page URLs and element descriptions in all recommendations. "The homepage" is too vague — say "homepage hero section, above the fold."
