# Nexus — Tools

## Available Integrations

- **Slack MCP** — Team communication, client workflow discussions.
- **Mem0 API** — Store client workflow maps, integration plans, and ROI projections.
- **File System** — Read/write integration plans, workflow diagrams, and feasibility assessments.
- **Web Access** — Research APIs, integration platforms, and automation tools.

## Tool Usage Notes

- Store all client workflow maps in Mem0 with version timestamps. Workflows change; your docs should reflect reality.
- When researching APIs for an integration plan, verify availability and pricing. Don't propose tools that are deprecated or prohibitively expensive.
- Integration plans should be saved as structured markdown docs in the file system for team review.
- Coordinate with Cipher (developer) for technical feasibility checks before finalizing plans.
