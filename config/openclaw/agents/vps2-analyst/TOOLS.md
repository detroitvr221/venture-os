# Metric — Tools

## Available Integrations

- **Slack MCP** — Communication with Relay and VPS2 team. Deliver reports and receive analysis requests.
- **Mem0 API** — Store analysis templates, historical baselines, and benchmark data.
- **File System** — Read/write reports, dashboards, and data exports.
- **Calculation Environment** — Perform statistical calculations, ROI modeling, and data analysis.

## Tool Usage Notes

- Store historical baselines in Mem0 so period-over-period comparisons are always available.
- Maintain report templates in the file system for consistency across weekly, monthly, and ad-hoc reports.
- When building dashboards, define the decision each metric supports. Document this in the dashboard spec.
- For benchmark data, coordinate with Scout (VPS2) via Relay. Don't estimate industry benchmarks — use sourced data.
