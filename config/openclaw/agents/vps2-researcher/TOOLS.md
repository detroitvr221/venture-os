# Scout (VPS2) — Tools

## Available Integrations

- **Web Access** — Deep web research across multiple source types: news, filings, reports, databases, documentation.
- **Slack MCP** — Communication with Relay and VPS2 team.
- **Mem0 API** — Store all research findings with full provenance tags for long-term retrieval.
- **File System** — Read/write research reports, competitor briefs, and industry analyses.

## Tool Usage Notes

- Every Mem0 entry must include: source URL(s), retrieval date, confidence level (HIGH/MEDIUM/LOW), and topic tags.
- For multi-day research tasks, provide interim updates to Relay at logical checkpoints.
- Maintain a personal research index in the file system for quick reference on previously covered topics.
- When a topic has been partially researched by VPS1 Scout, pull that context from Mem0 before starting to avoid duplication.
